##### A short python script that visualizes christmas songs locally in your terminal. No imports necessary.

## Setup

Assuming you already have python downloaded, start by cloning this repository:

```shell
git clone https://github.com/JummyJoeJackson/JummyJoeJackson/christmas-lyrics
cd christmas-lyrics
```

After cloning the repo, run:
```shell
pip install -r requirements.txt
python song.py
```

You should see the dazzling christmas tree changing colors in your console! The lyrics to Jingle Bells will appear below the tree.

To stop the program, hold down:
```shell
Ctrl + C
```
